{Constructors}

Constructor is block of code which can be used as the replacement of the method, there are certain rules to create a constructor 

1. Name of constructor will be same as of the class
2. Constructor gets called automatically at the time of instantiation of the class
3. Constructor doesnot have a return type
4. If we are not creating any constructor then java provides default constructor
